# API Schémata

Tato složka obsahuje OpenAPI schémata a specifikace API.

## 📁 Soubory

### `Event-API-v20-SNAPSHOT.yaml`
OpenAPI 3.0 specifikace pro Event API verze 2.0.

Tento soubor definuje:
- Všechny API endpointy
- Request/Response modely
- Validace a omezení
- Error responses

## 📖 Použití

Schéma můžete použít pro:
- Generování API dokumentace
- Validaci requestů
- Generování klientských SDK
- Testování API
